# Konglo OS Brand Assets — WebP Package

This package contains the complete Konglo OS raster brand collection generated with GPT Image and prepared for web/product use.

## Folder structure

- `00-original-generations/` — all 10 original generated artworks converted one-to-one to WebP.
- `01-logo/` — primary, stacked, wordmark, shield mark, transparent variants, and monochrome sheets.
- `02-product/` — app icons, favicon source sizes, desktop/mobile splash screens, and wallpapers.
- `03-marketing/` — Open Graph image, social profile, platform covers, and pitch-deck cover.
- `04-guidelines/` — full and web-size brand boards.
- `preview/` — contact sheet of the collection.
- `manifest.json` — dimensions, color mode, and SHA-256 for every asset.

## Important production notes

- Every image file is WebP. No SVG or PNG is included.
- Transparent WebP logo variants are background-removed derivatives of the GPT Image raster outputs.
- AI-generated typography can contain tiny visual inconsistencies. The main “Konglo OS” wording is readable in the selected outputs, but review all small labels before publishing.
- For UI use, prefer the files in `01-logo/` and `02-product/`; keep `00-original-generations/` as archival source artwork.
